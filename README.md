# Análisis de SVD y PCA en el Procesamiento de Imágenes

Este proyecto explora la aplicación de la **Descomposición en Valores Singulares (SVD)** y el **Análisis de Componentes Principales (PCA)**, fundamentales en el área de Ciencia de Datos.

## Contenido
- Compresión de imágenes mediante SVD.
- Reducción de dimensionalidad con PCA.
- Análisis teórico y práctico aplicado.

## Tecnologías utilizadas
- Python
- NumPy / Pandas
- Matplotlib (Visualización)
- Google Colab
